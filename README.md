# Vizaje-Nica Price Monitor

Internal price monitoring dashboard for Vizaje-Nica managers.

## Apps

- frontend — Next.js dashboard
- backend — NestJS API
- parser-worker — Playwright parser worker
- shared — shared types and schemas

## Main features

- manager brand lists
- product price comparison
- competitor links
- manual product update
- manual brand update
- night parser from 19:00 to 08:00
- PostgreSQL price history
